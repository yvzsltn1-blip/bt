# AGENTS.md

## Codex çalışma kuralları

- Superpowers workflow kullan.
- Önce kısa plan yap.
- Görevi küçük adımlara böl.
- Sadece gerekli dosyaları incele.
- Tüm repoyu tarama; önce ilgili dosyaları bul.
- Gereksiz refactor yapma.
- Kullanıcı istemedikçe yeni özellik ekleme.
- Mevcut davranışı bozma.
- Değişiklikleri minimum dosyada tut.

## Token / kota tasarrufu

- Kısa cevap ver.
- Uzun açıklama yazma.
- Büyük logları yapıştırma.
- Test çıktısını sadece özetle.
- Dosya içeriklerini komple tekrar etme.
- Markdown raporları kısa tut.
- Özet istenirse en fazla 5 madde yaz.
- İş bitince sadece şunları yaz:
  - Değişen dosyalar
  - Ne yapıldı
  - Test sonucu

## Kod değişikliği öncesi

- Önce ilgili dosyaları tespit et.
- Emin değilsen soru sormak yerine en makul varsayımla ilerle.
- Büyük değişiklik gerekiyorsa önce kısa plan yaz.
- Gereksiz terminal komutu çalıştırma.

## Kod değişikliği sonrası

- Değişen dosyaları listele.
- Uzun diff yapıştırma.
- Başarısız test varsa sadece hatanın özünü yaz.
- Sonraki adım önerisini kısa tut.
