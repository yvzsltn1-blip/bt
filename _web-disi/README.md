Bu klasor, webde canli gorunen uygulamanin ana akisina dahil olmayan gelistirme dosyalarini toplar.

- `docs`: notlar, arastirma ve oturum kayitlari
- `python`: masaustu/Colab Python araclari
- `analysis-js`: canli sayfalarda yuklenmeyen analiz scriptleri
- `tests`: test ve smoke test dosyalari
- `data-dumps`: dump, ara veri ve cikti dosyalari
- `scratch`: yedek veya gecici JS dosyalari
- `colab-workspace`, `scripts`, `standalone`, `sonuc-arsivi`, `local-backups`, `__pycache__`: canli web akisindan ayri calisma klasorleri
